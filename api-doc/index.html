Upload de Arquivos

POST /api/upload
Autenticação
Bearer Token necessário
Parâmetros (multipart/form-data)
file: File (arquivo para upload)
Tipos de Arquivo Permitidos
- image/jpeg
- image/png
- application/pdf
- text/plain
Limite de Tamanho
Máximo: 5MB
Resposta de Sucesso (200)
{
    "status": "success",
    "message": "Arquivo enviado com sucesso",
    "data": {
        "file_path": "/uploads/nome_do_arquivo.extensao"
    }
}
Resposta de Erro (400)
{
    "status": "error",
    "message": "Mensagem de erro específica",
    "data": null
}
Possíveis Mensagens de Erro
- "Nenhum arquivo foi enviado"
- "File type not allowed"
- "File exceeds the limit of 5 MB"
- "Failed to create the destination directory"
- "File already exists in the directory"
- "Failed to move the file"
